# MITS API

## Ruby version
- `2.3.1`


## Set up the Pow development server

- [docs](http://pow.cx/)

Install Pow

```bash
$ curl get.pow.cx | sh
```

Create a symlink to the Pow server

```bash
$ cd ~/.pow
$ ln -s /path/to/mits_api
```

Visit `http://mits_api.dev/`.
